**🛡️ SHEild – Smart Personal Safety Web App**

SHEild is a Progressive Web Application (PWA) designed to enhance personal safety by providing quick access to emergency tools, secure information storage, and safe zone awareness. The application is lightweight, mobile-friendly, and works offline.

It is built using pure HTML, CSS, and JavaScript with PWA support, making it installable like a mobile app.

**🌟 Features**

🔐 Secure Vault for storing important personal data

🚨 Emergency Access Page for quick action

📍 Safe Zones section for awareness and safety locations

📞 Emergency Contact Management

⚙️ Customizable Settings

👋 Onboarding screen for first-time users

📱 Installable as a Progressive Web App (PWA)

🌐 Offline support using Service Worker
